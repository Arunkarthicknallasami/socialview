All Types

com.hendraanggrian.socialview.Hashtag

A data class of hashtag to be used with com.hendraanggrian.widget.HashtagAdapter.
com.hendraanggrian.widget.HashtagAdapter

Default adapter for displaying hashtag in SocialAutoCompleteTextView. Note that this adapter is completely optional, any adapter extending android.widget.ArrayAdapter can be attached to SocialAutoCompleteTextView.
com.hendraanggrian.socialview.Mention

A data class of mention to be used with com.hendraanggrian.widget.MentionAdapter.
com.hendraanggrian.widget.MentionAdapter

Default adapter for displaying mention in SocialAutoCompleteTextView. Note that this adapter is completely optional, any adapter extending android.widget.ArrayAdapter can be attached to SocialAutoCompleteTextView.
com.hendraanggrian.widget.SocialAdapter

An ArrayAdapter customized with filter to display items. It is a direct parent of default HashtagAdapter and MentionAdapter, which are optional adapters.
com.hendraanggrian.widget.SocialAutoCompleteTextView

android.widget.MultiAutoCompleteTextView with hashtag, mention, and hyperlink support.
